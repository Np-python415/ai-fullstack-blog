# AI 功能说明

## 🤖 AI 摘要生成功能

本项目集成了 OpenAI API，实现了文章摘要生成功能，展示了完整的 **RAG（检索增强生成）工作流**。

### 功能特点

1. **RAG 工作流演示**
   - 知识检索：从数据库获取文章内容
   - 提示词工程：构建专业的提示词模板
   - AI 调用：调用 OpenAI API 生成摘要

2. **智能摘要生成**
   - 自动分析文章内容
   - 提取核心观点
   - 生成 2-4 句话的简洁摘要

3. **容错机制**
   - 如果没有配置 API Key，返回模拟结果（用于演示）
   - 完善的错误处理

### 使用方法

1. 打开任意文章详情页
2. 在文章顶部找到"AI 智能摘要"卡片
3. 点击"生成摘要"按钮
4. 等待 AI 分析并生成摘要

### 技术实现

- **API 路由**：`app/api/ai/summarize/route.ts`
- **前端组件**：`app/blog/[id]/page.tsx`
- **使用的模型**：GPT-3.5-turbo（可配置）

### 配置说明

在 `.env.local` 中配置：

```env
OPENAI_API_KEY=sk-xxxxx
```

如果没有配置，功能会返回模拟结果，仍然可以演示 RAG 工作流。
